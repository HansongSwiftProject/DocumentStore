ReadWriteTransaction

public class ReadWriteTransaction: ReadTransaction, ReadWritableTransaction

A transaction on a DocumentStore that can perform reads and writes.

  • add(document:)

    Add the given Document to the DocumentStore this ReadWriteTransaction is associated with.

    Precondition

    The DocumentDescriptor of the Document you are adding must be registered with the DocumentStore the given ReadTransaction is associated with. If this isn’t the case a TransactionError.documentStoreError is thrown the DocumentStoreError will be of kind documentDescriptionNotRegistered.

    Parameter

    Parameter document: Document to add to the DocumentStore

    Throws

    TransactionError on all failures
