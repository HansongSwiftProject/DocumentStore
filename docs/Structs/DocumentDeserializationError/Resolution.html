Resolution

public enum Resolution

The prefered resolution for the DocumentDeserializationError.

Note

deleteDocument will only persist the deletion if you commit the transaction with CommitAction.saveChanges.

Warning

abortOperation will make every Collection containing this particular document to fail with an error. Use skipDocument to not remove the data, but continue gracefully.

  • skipDocument: Skips the Document leaving it untouched in the store

  • deleteDocument: Removes the Document from the store

  • abortOperation: Abort the current operation
